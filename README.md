# Hyphantiké · Arquitectura Política

Plataforma web oficial del espacio de consultoría Hyphantiké, dedicada al análisis estructural de la trama política local, nacional e internacional. El sitio está diseñado con un enfoque minimalista, académico y de alta legibilidad, brindando acceso a servicios de consultoría y publicaciones de ensayos dinámicos.

🌐 **Sitio en vivo:** [https://rodriguezguillermo.github.io/hyphantike/](https://rodriguezguillermo.github.io/hyphantike/)

---

## ⚙️ Características Principales

*   **Sincronización Automatizada:** Integración directa con la API de Google Drive para renderizar automáticamente los últimos ensayos públicos en formato PDF sin necesidad de actualizar el código fuente.
*   **Diseño Editorial:** Interfaz elegante construida con una paleta de colores sobria y la tipografía clásica *EB Garamond* para garantizar una experiencia de lectura de prestigio.
*   **Enrutamiento Seguro:** Implementación de `HashRouter` a través de React Router para una navegación fluida (SPA) totalmente compatible con el alojamiento de GitHub Pages.
*   **Totalmente Responsivo:** Adaptación perfecta a dispositivos móviles, tablets y monitores de escritorio.

---

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** React (con TypeScript)
*   **Herramienta de Construcción:** Vite
*   **Enrutamiento:** React Router (v6+)
*   **Estilos:** CSS Vanilla / Tailwind CSS
*   **Íconos:** Lucide React
*   **Integraciones:** Google Drive API v3
*   **Despliegue:** GitHub Pages (`gh-pages`)
